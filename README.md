# 个人记账管理系统 python21

## 项目概述

个人记账管理系统是基于 Python 语言和 Django 框架开发的应用，旨在帮助用户高效管理个人财务。该系统具备详尽的部署文档和完整的设计报告，便于用户理解和使用。

## 技术栈

- **后端：** Python + Django
- **数据库：** SQLite（默认），亦支持 PostgreSQL、MySQL 等
- **前端：** HTML + CSS + JavaScript

## 功能模块

- **账户管理：** 管理用户及其权限
- **收入管理：** 记录和管理收入来源
- **支出管理：** 记录和管理支出项目
- **预算规划：** 设定和跟踪预算执行情况
- **报表统计：** 生成和导出财务报表

## 系统角色

- **普通用户：** 可访问和操作个人的财务数据
- **管理员：** 负责用户管理和系统维护

## 运行环境

- **推荐环境：** Python 3.8 或更高版本
- **操作系统：** 兼容 Linux、macOS 和 Windows

## 部署步骤

具体的部署步骤请参考项目提供的部署文档。

